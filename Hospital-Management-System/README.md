\# 🏥 Hospital Management System



\## Overview



A console-based Hospital Management System developed using Python.



The application helps manage patient records, search patients, update records, and generate hospital bills using Object-Oriented Programming concepts.



\---



\## Features



\- Add Patient

\- View Patients

\- Search Patient

\- Update Patient Details

\- Generate Bill

\- File Handling

\- Menu Driven Interface



\---



\## Technologies Used



\- Python

\- NumPy



\---



\## Concepts Used



\- Object-Oriented Programming (OOP)

\- Encapsulation

\- File Handling

\- Exception Handling

\- Loops

\- Functions



\---



\## Installation



Install NumPy



```bash

pip install numpy

```



Run the project



```bash

python hospital.py

```
